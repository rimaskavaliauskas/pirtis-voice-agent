# SKILL: Pirties projektavimo dialogas

## Metodologija efektyviam pirties projektų valdymui

**Versija:** 3.1 (galutinė)
**Atnaujinta:** 2026-01-12
**Pagrindas:** 15 metų pirties projektavimo patirtis, Pirties Laboratorijos metodika

---

## 1. SKILL APŽVALGA

### Paskirtis

Šis skill skirtas struktūruotam pirties projektavimo procesui – nuo pirminio pokalbio su užsakovu iki detalios projektavimo užduoties parengimo.

### Pagrindinis principas

**Projektavimo užduotis turi būti konkreti ir aiški užsakovui.**

- Detales ir variantus laikyk skill'e (agento kontekste)
- Užsakovui pateik rekomenduojamus sprendimus
- Jei sprendimas nėra aiškiai patvirtintas – įtrauk į klausimų sąrašą failo pabaigoje

## 2. INFORMACIJOS SURINKIMO SCHEMA

### Pirmas kontaktas – VIZIJA

**Pradėk nuo vizijos, ne nuo technikalijų!**

```
PIRMAS KLAUSIMAS (visada):
"Papasakokite, kaip įsivaizduojate idealų pirties seansą –
kas dalyvautų, ką veiktumėte, kiek laiko praleistumėte?"
```

### Kritiniai parametrai

| Klausimas                 | Kodėl svarbu               | Tipinės vertės              |
| ------------------------- | -------------------------- | --------------------------- |
| Ploto limitas             | Visų sprendimų pagrindas   | 15-50 kv.m vidaus           |
| Žmonių skaičius           | Garinės dydis, plautai     | 2-4 / 4-8 / 8+              |
| Naudojimo tipas           | Konstrukcijos pasirinkimas | Šeimos / Verslo / Mišrus    |
| Krosnies tipas            | Mikroklimato pagrindas     | Periodinio / Nuolatinio     |
| **Statybos technologija** | Lemia daugelį sprendimų    | Rąstai / Karkasinė / Mūrinė |

### SVARBU: Statybos technologija

**Kuo anksčiau išsiaiškinta – tuo tiksliau projektuojama.**

Galimos technologijos (agento žinojimui, NE siūlyti užsakovui be prašymo):

| Technologija           | Kūrenimo laikas | Ypatybės                       |
| ---------------------- | --------------- | ------------------------------ |
| **Rąstai (rankiniai)** | 5+ val.         | Autentiška, susitraukimas      |
| **Rąstai (frezuoti)**  | 4-5 val.        | Sandariau, mažiau autentiška   |
| **Rąstai (džiovinti)** | 4-5 val.        | Mažesnis susitraukimas         |
| **Karkasinė**          | 1-2 val.        | Greitas paruošimas             |
| **Mūrinė**             | 3-4 val.        | Ilgaamžė, specifiniai įgūdžiai |
| **Bačka**              | ~1 val.         | Ekonomiška, ribota erdvė       |

**TAISYKLĖ:** Alternatyvias technologijas siūlyk TIK kai užsakovas to pageidauja arba dar neapsisprendęs.

## 3. GARINĖS PROJEKTAVIMAS

### Plautai – pagrindinis garinės elementas

**Kritinis klausimas (agento žinojimui):**

```
"Kas jums svarbiau garinėje – sėdimos vietos bendravimui
ar gulimos vietos procedūroms?"
```

**SVARBU:** Šio klausimo NEDĖK į projektavimo užduotį. Jį užduok dialogo metu ir rezultatą fiksuok kaip faktą (pvz., "prioritetas – gulimos vietos").

| Prioritetas        | Implikacijos                                |
| ------------------ | ------------------------------------------- |
| **Sėdimos vietos** | Daugiau žmonių, mažiau erdvės kiekvienam    |
| **Gulimos vietos** | Mažiau žmonių, pilnas komfortas procedūroms |

### Durys

**Rekomenduojamas sprendimas:** Tradicinės žemos durys (neišleidžia garo, graži tradicinėje pirtyje)

**Jei užsakovas nori aukštų durų:** Įtrauk į klausimų sąrašą patvirtinimui

**Antros durys (pvz., į ežerą):** Jei užsakovas mini – užfiksuok kaip pageidavimą, įtrauk į klausimus

### Apšvietimas

**4 zonos, kurias būtina apšviesti:**

| Zona                  | Kodėl būtina              |
| --------------------- | ------------------------- |
| Išėjimas              | Saugumas (nepaslysti)     |
| Karšto vandens vieta  | Apsauga nuo nusiplikinimo |
| Prie krosnies         | Garo pilimas              |
| Pirtininko darbo zona | Vanojimo kokybė           |

+ Tvarkymo apšvietimas (po plautais arba bendras)
+ Reguliavimas (procedūrų metu pritemdytas)

**Užduotyje:** Rašyk trumpai – "reguliuojamas apšvietimas keturiose zonose"

### Ventilacija – KRITIŠKAI SVARBU

**Ventiliacijos tipai (agento žinojimui):**

| Tipas           | Aprašymas                      |
| --------------- | ------------------------------ |
| Per priverstinę | Mechaninė ventiliacija         |
| Per kanalą      | Atskiras ventiliacijos kanalas |
| Per kaminą      | Išnaudojama kamino trauka      |
| Tiesiai į lauką | Anga sienoje                   |

**Šviežio oro padavimas:**

- Per krosnį (sušildomas)
- Atskiras kanalas

**Lokali ventilacija:** Prie gulinčio žmogaus nosies vanojimo metu

**Gūsinė ventilacija:** Langas atidaromas tarp užėjimų, uždaromas procedūrų metu

**KRITINĖ TAISYKLĖ:** Ventiliacijos išėjimas turi būti ATSKIRAS nuo poilsio patalpos!

**Užduotyje:** Rašyk trumpai ir konkrečiai – "ventilacija su atskiru kanalu nuo poilsio patalpos, reguliavimo galimybė"

## 4. POILSIO PATALPOS PROJEKTAVIMAS

### Šildymas – BŪTINAS

**DĖMESIO:** Šilumos iš garinės per duris NEUŽTENKA! Be papildomo šildymo patalpa bus šalta ir drėgna.

**Šildymo variantai (agento žinojimui):**

| Variantas             | Privalumai             | Trūkumai          |
| --------------------- | ---------------------- | ----------------- |
| Šildomos grindys      | Patogu, nematoma       | Lėtas reagavimas  |
| Elektrinis šildytuvas | Greitas, paprastas     | Nėra estetikos    |
| Malkinė krosnelė      | Ugnis, tradicija       | Reikia prižiūrėti |
| Židinys               | Estetika, ugnis        | Vieta, kaminas    |
| Mass Heat Storage     | Ilgas šilumos laikymas | Sudėtinga         |

**Svarbus klausimas:** Ar šildymas tik techninis, ar norima ESTETINĖS UGNIES?

**Užduotyje:** Nerašyk visų variantų! Rašyk konkrečiai:

- Jei žinoma: "Šildomos grindys (jau yra)"
- Jei nežinoma: Įtrauk į klausimų sąrašą

### Arbatos zona

**Variantai:**

- Tik virdulys
- Mini virtuvėlė
- Kriauklė indams

**Užduotyje:** Jei neaišku – įtrauk į klausimus

## 5. TERASOS

### Techninė terasa (pirtininkui)

**SVARBU:** Pirtininkas NERUOŠIA maisto! Daugiausiai arbatą.

Nerašyk "maisto ruošimo zona" – tai klaidingas signalas užsakovui.

### Svečių terasa

**Klausimai aptarimui:**

- Ar po stogu, ar atvira?
- Sėdimos/užkandžiavimo ar gulimos vietos?
- Ar bus ugnies zona? (dujų šildytuvas, šašlykinė, grilis)

## 6. IŠORINĖ PIRTIES INFRASTRUKTŪRA

### SVARBU: Visada užduok šiuos klausimus!

Išorinė infrastruktūra dažnai pamirštama, bet ji labai svarbi pilnai pirties patirčiai. **Būtinai paklausk dialogo metu:**

### Karšto vandens zona

**Klausimas:**

```
"Ar pageidaujate karšto vandens zonos lauke – kubilo
arba mūrinio baseinuko (rotenburo tipo)?"
```

**Variantai (agento žinojimui):**

| Tipas                         | Aprašymas               | Šildymas                           |
| ----------------------------- | ----------------------- | ---------------------------------- |
| **Standartinis kubilas**      | Medinis, perkamas       | Atskiras šildytuvas                |
| **Rotenburo tipo baseinukas** | Individualiai mūrijamas | Nuo pirties krosnies arba atskiras |
| **Lietas betoninis**          | Individualus projektas  | Įvairūs variantai                  |

**Jei užsakovas nori rotenburo:**

- Vieta ir orientacija?
- Šildymas nuo pirties krosnies ar atskiras?
- Talpa (kiek žmonių)?
- Naudojimas žiemą?

### Šalto vandens zona

**Klausimas:**

```
"Ar pageidaujate šalto vandens kubilo pasinėrimui
po garinės?"
```

**Klausimai jei taip:**

- Talpa?
- Vandens keitimo dažnumas (po kiekvieno karto / filtruojamas)?
- Vėsinimas vasarą?

### Lauko dušai

**Klausimas:**

```
"Ar pageidaujate lauko dušų – šilto ir/ar šalto?"
```

**Klausimai jei taip:**

- Vieta (prie išėjimo į ežerą / prie kubilų)?
- Konfigūracija (atskiri šiltas+šaltas / kombinuotas)?
- Šilto vandens šaltinis (nuo pirties boilerio)?

### Užduotyje

Jei užsakovas pageidauja išorinės infrastruktūros – **sukurk atskirą skyrių "IV. IŠORINĖ PIRTIES INFRASTRUKTŪRA"** su:

- Karšto vandens zona (tipas, šildymas)
- Šalto vandens zona
- Lauko dušai

Neišsiaiškintus klausimus perkelk į "Klausimai aptarimui" skyrių.

## 7. DOKUMENTACIJOS STRUKTŪRA

### Projektavimo užduoties šablonas (v3.1)

```markdown
# [Užsakovo vardas] pirties projektavimo užduotis

## I. PROJEKTO SANTRAUKA
- Literatūrinė pastraipa apie viziją (~10 sakinių)
- Esminių parametrų lentelė
- Naudojimo scenarijai

## II. STATYBOS TECHNOLOGIJA
- Pasirinkta technologija
- Implikacijos projektui

## III. PATALPŲ SĄRAŠAS
- Garinė (plautai, durys, apšvietimas, ventilacija, vanduo)
- Poilsio patalpa (arbatos zona, šildymas)
- Persirengimo zona
- Tualetas
- Išorės erdvės (terasos, papildomi pastatai)

## IV. IŠORINĖ PIRTIES INFRASTRUKTŪRA ← JEI AKTUALU
- Karšto vandens zona (rotenburas / kubilas)
- Šalto vandens zona (kubilas)
- Lauko dušai

## V. TECHNINIAI SPRENDIMAI
- Krosnis
- Ventiliacijos sistema

## VI. PROJEKTO ADEKVATUMO ĮVERTINIMAS
- Stipriosios pusės
- Atitikimas poreikiams
- Įgyvendinamumo vertinimas

## VII. RIZIKŲ ANALIZĖ
- Kritinės
- Vidutinės

## VIII. KLAUSIMAI APTARIMUI ← SVARBU!
- Sprendimai, reikalaujantys patvirtinimo
- Išorinė infrastruktūra (jei neišsiaiškinta)
- Būtina išsiaiškinti prieš projektavimą

## IX. PROJEKTAVIMO DARBŲ APIMTIS
- Projektavimo eiga
- Orientaciniai terminai
```

### "Būtina aptarti" taisyklė

**NIEKADA nerašyk "Būtina aptarti:" teksto vidury!**

Jei sprendimas:

1. Netradicinis (pvz., antros durys į ežerą)
2. Nepatvirtintas užsakovo
3. Turi kelis variantus

→ PERKELK į skyrių "VIII. KLAUSIMAI APTARIMUI"

---

## 8. PROJEKTAVIMO DARBŲ EIGA

### Standartinė eiga (įtrauk į užduotį)

**1. Sutarties etapas**

- Patvirtinama projektinė užduotis
- Projektuotojas pateikia kainą
- Sutariama, gaunamas avansas

**2. Eskizinis projektavimas**

- 2-3 skirtingi variantai (planai, sienos, funkcijos)
- Užsakovas pasirenka vieną
- Dialogas, korekcijos
- Variantas patvirtinamas

**3. Detalus projektavimas**

- Trimatis pilnas projektas
- Detalumas pagal sutartį

**4. Įforminimas (jei reikia)**

- Daugeliui pirčių nereikia oficialaus tvirtinimo
- Jei reikia – medžiaga perduodama architektui

## 12. DIALOGŲ PAVYZDŽIAI

### Pirmo kontakto pradžia

```
AGENTAS: Sveiki! Prieš kalbant apie techninius dalykus,
norėčiau suprasti jūsų viziją. Papasakokite – kaip
įsivaizduojate idealų pirties vakarą?
```

### Plautų prioritetas (dialogo metu)

```
AGENTAS: Kalbant apie garinę – man svarbu suprasti,
kas jums prioritetas.

Ar norite daugiau SĖDIMŲ vietų – kad tilptų daugiau
žmonių bendrauti? Ar GULIMŲ vietų – kad būtų patogu
priimti procedūras?
```

### Išorinė infrastruktūra

```
AGENTAS: Dar svarbus klausimas apie lauko zoną.

Ar pageidaujate karšto vandens kubilo arba mūrinio
baseinuko lauke? Daugelis mėgsta po garinės
pasimėgauti karštu vandeniu.

O kaip su šalto vandens kubilu pasinėrimui?

Ir dar – ar norėtumėte lauko dušų prie pirties?
```

### Rotenburo šildymas

```
AGENTAS: Dėl karšto vandens baseinuko – ar norėtumėte
jį šildyti nuo pirties krosnies? Tai ekonomiška, bet
reikalauja sudėtingesnės sistemos.

Alternatyva – atskiras šildytuvas, paprasčiau, bet
papildomos išlaidos.
```
